# Prompts

18 pre-built workflows that guide the AI through multi-step tasks.

| Prompt | Description |
|--------|-------------|
| `find-and-play` | Search for media by name and start playback on a connected client |
| `resume-watching` | Pick up where you left off on in-progress movies, episodes, and audio |
| `whats-new` | Recently added media and next episodes — a personalized viewing guide |
| `movie-night` | Curated movie suggestions filtered by genre and rating |
| `music-listen` | Find and play music by artist, album, or song; generate a smart mix |
| `binge-watch` | Check progress on a TV series and queue up next episodes |
| `fix-subtitles` | Search for and download missing subtitles for a movie or episode |
| `who-is-watching` | Dashboard of all active playback sessions |
| `troubleshoot` | Diagnose server issues — logs, tasks, plugins, system status |
| `bulk-metadata-fix` | Find and fix metadata issues across a library |
| `subtitle-audit` | Audit a library for missing subtitles and batch-download them |
| `library-report` | Library analytics: stats, codecs, unplayed items, recent additions, duplicates |
| `duplicate-finder` | Find duplicate media items and optionally remove inferior copies |
| `watch-history` | View watch history for a user over a time period |
| `codec-optimize` | Analyze media codecs and optimize transcoding settings |
| `parental-controls` | Set up kid-safe access with content rating limits and library restrictions |
| `server-setup` | Review and optimize server configuration |
| `library-health` | Server health check: status, storage, tasks, plugins, logs, and backups |
